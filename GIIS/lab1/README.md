# Лабораторная работа №1 — Алгоритмы построения отрезков

**Дисциплина:** Графический интерфейс интеллектуальных систем (GIIS)

Графический редактор, реализующий построение отрезков тремя алгоритмами:

- ЦДА (цифровой дифференциальный анализатор)
- Целочисленный алгоритм Брезенхема
- Алгоритм Ву (anti-aliasing)

## Запуск

```bash
cd GIIS/lab1
pip install -r requirements.txt
python main.py
```

> Требуется Python 3.7–3.12 (PyQt5 не поддерживает 3.13+).

## Использование

- **Выбор алгоритма** — меню «Отрезки» или кнопки на панели инструментов
- **Пользовательский режим** — рисование мышью (два клика) или ввод координат + «Построить»
- **Отладочный режим** — пошаговая визуализация на дискретной сетке с таблицей параметров
